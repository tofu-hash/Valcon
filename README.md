# Valcon

Конвертер валют в telegram

![logo.jpg](source/screenshots/logo.jpg)

# Запуск

Установи нужные библиотеки:
```bash
pip3 install -r requirements.txt
```

Установи `API_KEY` в `config.py`:
```python
# Config.py

API_KEY = '62843...fO70g'
```

Запусти бота (`run.py`):
```bash
cd Valcon
```
```bash
python3 run.py
```
